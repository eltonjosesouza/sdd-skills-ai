# Security Policy

## Supported Versions

Only the latest version of `sdd-skills-ai` is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not report it publicly** via GitHub issues.

Instead, please report it privately by contacting [eltonjosesouza](https://github.com/eltonjosesouza).

Once reported, we will:
1. Acknowledge your report.
2. Investigate and verify the vulnerability.
3. Provide a timeline for a fix.
4. Notify you once the vulnerability is addressed.
